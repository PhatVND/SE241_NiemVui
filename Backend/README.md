# Cấu hình file .env

```
PORT=
MONGODB_LINK=
SECRET_KEY_ACCESS=
EXPIRE_ACCESS=
SECRET_KEY_REFRESH=
EXPIRE_REFRESH=
```

- Lưu ý: Refresh Access phải có thời gian tồn tại lâu hơn access
- Secret key là 1 chuỗi 32 byte để làm hash key cho jwt
